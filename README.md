# Personal Finance Snapshot

## Overview
A simple finance tracker that helps users monitor income, expenses, and budgets.

## Features
- Add income and expense transactions
- Spending visualization with charts
- Budget tracking per category
- Persistent data using LocalStorage

## Tech Stack
- React
- TypeScript
- TailwindCSS
- Recharts

## Design Decisions
I focused on clarity and quick financial insights.
Charts help users instantly understand spending patterns.

## Challenges
Managing LocalStorage state and ensuring charts update dynamically.

## Improvements (Future)
- Authentication
- Cloud storage
- Recurring transactions
- Mobile optimization
